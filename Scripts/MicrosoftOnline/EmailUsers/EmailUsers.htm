We will begin transitioning your email to our new servers as previously communicated. With this transition, you must complete the following tasks immediately (in order):

NOTE: It is very important that you perform all of the steps described below ASAP to reduce disruption of your email.

STEP #1 - Initiate Migration

You will shortly be receiving an email with a link requesting your user name and password to your existing email account. Click on the link provided and enter your current user name and password in the corresponding fields. One you provide this information, your mailbox will be moved to our new system.

IF YOU DO NOT PERFORM THIS, YOUR OLD EMAIL WILL NOT BE ACCESSIBLE FROM OUR NEW SYSTEM.

STEP #2 - Install Applications

  1. Install Microsoft.NET Framework

    • Mac Users: No action required
  2. Install Office Communicator 2007
    • Windows Users: http://www.microsoft.com/online/occlient/en-us/default.mspx
    • Mac Users: No action required
  3. Install Office Live Meeting 2007 Client
    • Windows Users: http://office.microsoft.com/en-us/help/HA101733831033.aspx
    • Mac Users: No action required
  4. Install Conferencing Add-in for Microsoft Office Outlook
    • Windows Users: http://office.microsoft.com/en-us/help/HA102368901033.aspx
    • Mac Users: No action required
  5. Install Microsoft Exchange Online Connector for Office Outlook 2003 (Only required if running Outlook 2003)
    • Windows Users: http://www.microsoft.com/downloads/en/details.aspx?displaylang=en&FamilyID=604a539a-5ec1-4fb2-a3cf-60076aa17618
    • Mac Users: No action required
  6. Install Microsoft Online Services Sign In Application
    • Windows Users: http://www.microsoft.com/downloads/details.aspx?familyid=5c2ca866-4107-4ae5-98d5-76bf1b18ff87&displaylang=en

STEP #3 - Configure Computer

You will be issued a new user name and password when accessing our new system.
